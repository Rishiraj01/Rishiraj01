## Open Source Projects

### [B.A.T.H.O](https://github.com/sageoz/Batho) — Code Intelligence Engine
Open-source tool that transforms codebases into queryable, time-aware structured graphs using AST traversal and hypergraph extraction. Acts as a durable memory layer for AI agents, with token-budgeted context generation and incremental patching.

**Highlights:**
- Multi-language AST extraction via Tree-sitter (functions, classes, imports, calls, definitions)
- Structured graph generation with compressed output modes (4K–40K tokens) optimized for LLM context injection
- Time Machine: timestamped snapshots with entity/relationship diffs and staleness scoring
- Incremental patching with full audit trail and cherry-pick support
- Automatic tech stack detection across Python, Node.js, Java, .NET, Go, Rust, and more

### [KOB Monitoring Agent](https://github.com/Rishiraj01/KOB-Monitoring-Agent) — AI-Powered K8s Monitoring
An AI-powered monitoring agent for Kubernetes on Bare Metal clusters. Uses RAG and LLMs to diagnose cluster issues, alert humans, or attempt automated fixes.

**Highlights:**
- LLM-powered diagnosis (OpenAI/Gemini) enhanced with Kubernetes documentation via RAG
- Proactive monitoring of pod and node health
- RabbitMQ alerting pipeline for notification system integration
- Optional auto-remediation for well-defined issues
- FastAPI status endpoint, containerized for K8s or standalone deployment
